# Grades to Letters 

A simple C++ console application that takes grades (marks) for five different subjects, validates the input to ensure they fall within a realistic percentage range (0–100), and calculates the corresponding average and letter grade.

## Features
* **Multi-Subject Input:** Accepts integer marks for 5 separate subjects.
* **Input Validation:** Automatically detects invalid inputs (marks above 100 or below 0) and safely terminates with a `"Bad news!"` message.
* **Precise Calculations:** Prepared to handle decimal-point rounding for averages using the `<iomanip>` library.
